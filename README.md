# 386-Style Components for Angular

Angular/386 is a Twitter bootstrap components for Angular v.2+ to make webpages look like they are from the 1980s.

### Project status
Angular/386 is currently in alpha and under active development.
During alpha, new features will be added regularly and APIs will evolve based on user feedback.

If you'd like to contribute, you must follow our [contributing guidelines](https://github.com/angular/angular386/blob/master/CONTRIBUTING.md).
You can look through the issues (which should be up-to-date on who is working on which features
and which pieces are blocked) and make a comment.

High level stuff planned for Q1 2018 (January - March):
* RC and new release
* Research and exploration for data visualization
* Button
* Input
* List

## The goal of Angular/386
Our goal is to build a set of high-quality UI components built with Angular and TypeScript,
following the best practices of 1980's. These
components will serve as an example of how to write Angular code following best practices.

## Browser and screen reader support
Angular/386 supports the most recent two versions of all major browsers:
Chrome (including Android), Firefox, Safari (including iOS), and IE11 / Edge
